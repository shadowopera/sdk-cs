---
title: 'MinMaxExtensions'
---

Namespace: Shadop.Archmage.Sdk

Provides `Sample` extension methods that draw a uniform random value from a
 [MinMax<T>](../shadop-archmage-sdk-minmax-1/) range using a [Random](https://docs.microsoft.com/en-us/dotnet/api/system.random) source. The returned
 value lies in `[Min, Max]` (both ends included).

```csharp
public static class MinMaxExtensions
```
