Namespace: Shadop.Archmage

Implements the [IFS](../../sdk-cs/shadop-archmage-ifs/) interface to load files via Unity Resources.
Paths are resolved relative to any `Resources` folder; file extensions are stripped automatically.

```csharp
public class UnityResourcesFS : IFS
```

<Aside type="caution">
  Synchronous loading (`ReadAllBytes`) must be called from the Unity main thread.
</Aside>
